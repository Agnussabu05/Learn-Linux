Linux Command: rmdir

Description

The rmdir command is used to remove empty directories in Linux. If the directory contains any files or subdirectories, the command will fail.

Purpose and Usage

The rmdir command is primarily used to clean up empty directories. It ensures that only empty directories are deleted, making it safe for basic cleanup operations.

Usage: rmdir [directory]

Examples

Basic Usage: Remove an empty directory named example_dir:

rmdir example_dir

Visual Examples

Additional Information

The rmdir command is straightforward and limited to empty directories, making it a safe option for cleaning up specific parts of the file system. For directories containing files, use the rm -r command instead.
