Linux Command: grep

Description

The grep command searches for patterns within files.

Purpose & Usage

Purpose: To filter and locate specific text in files.

Usage: Ideal for log analysis and text processing.

Examples

1. Case-Insensitive Search:

grep -i "pattern" filename

2. Count the Number of Matches:

grep -c "pattern" filename

3. List File Names with Matches:

grep -l "pattern" *.txt

4. Match Whole Words Only:

grep -w "word" filename

5. Show Only Matching Part of the Line:

grep -o "pattern" filename

Explanation

Common Use Case: Debugging and extracting relevant information from logs.

Importance: A powerful text search tool for administrators and developers.
