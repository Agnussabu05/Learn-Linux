Linux Command: cat

Description

The cat command is used to display the contents of a file, create new files, or concatenate and view multiple files.

Purpose & Usage

Purpose: To quickly view, create, or manipulate text files.

Usage: Frequently used for displaying file contents, merging files, or adding line numbering to files.

Examples

1. Create a New File:

cat > newfile.txt
(Type content and press Ctrl+D to save.)

2. Display File Contents:

cat file.txt

3. Suppress Repeated Empty Lines:

cat -s file.txt

4. Number All Lines in a File:

cat -n file.txt

5. Show End-of-Line Characters:

cat -E file.txt

6. Display File Contents in Reverse Order:

tac file.txt

Explanation

Common Use Case: Viewing, creating, and combining text files. The tac command is a reverse of cat that displays file contents from bottom to top.

Importance: A versatile command for simple text file operations, often used in shell scripting and quick inspections.
