Linux Command: cp

Description

The cp command is used to copy files and directories from one location to another. It is commonly used for duplicating files, creating backups, or transferring files between directories.

Purpose and Usage

The cp command is a versatile tool for maintaining copies of files and directories.

Usage: cp [options] source destination

Examples

Basic Usage: Copy a file named file.txt to backup.txt:

cp file.txt backup.txt

Visual Examples

Additional Information

The cp command provides various options to enhance its functionality. For example:

  • -i: Prompts before overwriting files.
  • -u: Copies files only if the source file is newer than the destination file or if the destination file does not exist.
  • -v: Displays the details of the copying process.
