Linux Command: mkdir

Description

The mkdir command is used to create new directories.

Purpose & Usage

Purpose: To organize files by creating folders.

Usage: Often used for project or data organization by creating folders in the file system.

Examples

1. Create a New Directory:

mkdir new_folder
